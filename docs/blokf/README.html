blokf - README

Sistem Pengamanan Lebaran Blok F RT 024

Selamat datang di dokumentasi aplikasi Sistem Pengamanan Lebaran untuk Blok F RT 024.

Tentang Aplikasi

Aplikasi ini dirancang untuk memudahkan pengelolaan keamanan Lebaran di lingkungan Blok F RT 024 dengan menggunakan sistem QR Code. Petugas jaga dapat memindai (scan) QR code warga/tamu yang masuk atau keluar lingkungan.

Fitur Utama

Untuk Petugas Jaga (Mobile App)

  • Scan QR Code - Pindai QR dengan kamera untuk tracking masuk/keluar
  • Riwayat Scan - Lihat semua scan yang sudah dilakukan
  • Pengumuman - Terima pengumuman penting dari admin
  • Offline Mode - Tetap bisa scan tanpa internet
  • Auto Sync - Sinkronisasi otomatis saat ada internet

Untuk Admin (Dashboard)

  • Manajemen QR Code - Buat QR code untuk setiap rumah/blok
  • Manajemen Petugas - Kelola data petugas jaga
  • Manajemen Pos - Kelola pos-pos jaga
  • Pengumuman - Buat dan kirim pengumuman ke petugas
  • Monitoring Real-time - Pantau scan masuk/keluar
  • Laporan - Export laporan dalam format Excel

Kebutuhan Sistem

Aplikasi Mobile (Android)

  • Android: Versi 7.0 (Nougat) atau lebih tinggi
  • Kamera: Diperlukan untuk scan QR
  • Internet: Disarankan untuk sync data
  • Penyimpanan: Minimal 50 MB

Dashboard Admin

  • Browser: Chrome, Firefox, Safari, atau Edge (versi terbaru)
  • Internet: Diperlukan untuk akses dashboard

Cara Install

Untuk Petugas Jaga

  1. Download file APK blokf.apk dari link di bawah
  2. Buka file APK di Android Anda
  3. Jika muncul peringatan "Unknown Sources", klik "Settings" dan aktifkan "Allow from this source"
  4. Klik "Install" dan tunggu proses selesai
  5. Buka aplikasi "Pengamanan Lebaran"
  6. Login dengan PIN yang diberikan admin
  7. Pilih nama petugas dan pos tempat bertugas

Untuk Admin

  1. Kunjungi dashboard: https://rt24.hadirapp.com
  2. Login dengan kredensial admin Anda
