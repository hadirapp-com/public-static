## 1. PENGANTAR

Selamat datang di Panduan Penggunaan Aplikasi Mobile Sistem Pengamanan Lebaran Blok F RT 024.

Aplikasi ini digunakan oleh petugas jaga untuk memindai (scan) QR code warga/tamu yang masuk atau keluar lingkungan.

### Fitur Utama:

✅ **Scan QR Code** - Pindai QR dengan kamera
✅ **Riwayat Scan** - Lihat scan yang sudah dilakukan
✅ **Pengumuman** - Terima pengumuman penting
✅ **Offline Mode** - Tetap bisa scan tanpa internet
✅ **Auto Sync** - Sinkronisasi otomatis saat ada internet

### Kebutuhan Sistem:

- **Android:** Versi 7.0 (Nougat) atau lebih tinggi
- **Kamera:** Diperlukan untuk scan QR
- **Internet:** Disarankan untuk sync data
- **Penyimpanan:** Minimal 50 MB

## 2. INSTALASI APLIKASI

### Cara Install:

#### Opsi 1: Dari File APK

1. Minta file APK dari admin
2. Buka file **`pengamanan.apk`**
3. Jika muncul peringatan "Unknown Sources", klik **"Settings"** dan aktifkan **"Allow from this source"**
4. Klik **"Install"**
5. Tunggu proses instalasi selesai
6. Klik **"Buka"**

#### Opsi 2: Dari Play Store

1. Buka Google Play Store
2. Cari: **"Pengamanan Lebaran"**
3. Klik **"Install"**
4. Tunggu proses instalasi selesai

## 3. LOGIN PERTAMA

### Langkah-langkah Login:

1. Buka aplikasi **"Pengamanan Lebaran"**

2. Halaman PIN akan muncul

3. Masukkan **6-digit PIN** yang diberikan admin:
   ```
   PIN: 123456
   ```

4. Klik tombol **"Masuk"**

5. Setelah berhasil, Anda akan diminta memilih:
   - **Petugas Jaga:** Pilih nama Anda
   - **Pos Jaga:** Pilih pos tempat Anda bertugas

6. Klik **"Simpan"**

### Ganti Petugas/Pos:

Jika berganti pos atau petugas:

1. Klik menu **"Settings"** (⚙️) di pojok kanan atas
2. Klik **"Pilih Petugas"**
3. Pilih petugas baru
4. Klik **"Pilih Pos"**
5. Pilih pos baru
6. Klik **"Simpan"**

## 5. SCAN QR CODE

### Persiapan Sebelum Scan:

Sebelum melakukan scan:

1. ✅ **Pastikan sudah memilih petugas dan pos** yang benar
2. ✅ **Buka kamera** dan arahkan ke QR code
3. ✅ **Pastikan QR code terbaca jelas**
4. ✅ **Pilih tipe scan:** Masuk atau Keluar

### Cara Scan QR:

#### Langkah 1: Buka Menu Scan

1. Klik tab **"Scan"** di bagian bawah
2. Izinkan akses kamera jika diminta
3. Kamera akan aktif otomatis

#### Langkah 2: Arahkan ke QR Code

1. Arahkan kamera ke QR code yang ada di kartu/stiker
2. QR code harus berada di dalam kotak frame
3. Tunggu sampai QR terdeteksi

#### Langkah 3: Pilih Tipe Scan

Setelah QR terdeteksi:

1. Pilih tipe scan:
   - **🔽 MASUK** - Untuk tamu/warga yang MASUK lingkungan
   - **🔽 KELUAR** - Untuk tamu/warga yang KELUAR lingkungan

2. Klik tombol **"SCAN"**

#### Langkah 4: Konfirmasi Scan

Jika scan berhasil:

```
┌─────────────────────────────────┐
│  ✅ SCAN BERHASIL               │
├─────────────────────────────────┤
│                                 │
│  QR: Block A-101                │
│  Penanggung Jawab:              │
│  Bpk. Ahmad                     │
│                                 │
│  Tipe: MASUK                    │
│  Waktu: 13-03-2026, 18:30       │
│                                 │
│  [OK]                           │
└─────────────────────────────────┘
```

Klik **"OK"** untuk kembali ke scanner

#### Langkah 5: Scan Berikutnya

Anda bisa langsung scan QR berikutnya tanpa perlu kembali ke menu utama.

### Jika Scan Gagal:

```
┌─────────────────────────────────┐
│  ❌ SCAN GAGAL                  │
├─────────────────────────────────┤
│                                 │
│  QR Code tidak valid atau       │
│  tidak aktif                    │
│                                 │
│  [OK]                           │
└─────────────────────────────────┘
```

**Penyebab gagal:**
- QR code tidak terdaftar
- QR code sudah tidak aktif (melewati masa berlaku)
- QR code salah/korupsi

### Tips Scan yang Baik:

💡 **Pencahayaan Cukup** - Pastikan area terang
💡 **Fokus Kamera** - Tunggu sampai QR jelas
💡 **Jarak Tepat** - Jangan terlalu dekat atau jauh
💡 **Stabilkan Tangan** - Hindari goyangan saat scan

## 6. RIWAYAT SCAN

Riwayat scan menampilkan semua scan yang sudah Anda lakukan.

### Membuka Riwayat:

1. Klik tab **"Log"** di bagian bawah
2. Daftar semua scan akan muncul

### Informasi yang Ditampilkan:

```
┌────────────────────────────────────┐
│  📋 RIWAYAT SCAN                   │
├────────────────────────────────────┤
│                                    │
│  13 Maret 2026, 18:30              │
│  Block A-101                       │
│  Penanggung Jawab: Bpk. Ahmad      │
│  Petugas: Anda                     │
│  Pos: Pos 1 Utara                  │
│  Tipe: MASUK ✅                    │
│                                    │
│  13 Maret 2026, 17:45              │
│  Block B-205                       │
│  Penanggung Jawab: Ibu Siti        │
│  Petugas: Anda                     │
│  Pos: Pos 1 Utara                  │
│  Tipe: KELUAR 🔼                   │
│                                    │
└────────────────────────────────────┘
```

### Ikon Status:

| Ikon | Status |
|-----|--------|
| ✅ **Synced** | Scan sudah disimpan ke server |
| ⏳ **Pending** | Scan belum disinkronkan (offline) |

### Refresh Data:

Tarik layar ke bawah (pull to refresh) untuk memuat data terbaru.

### Hapus Riwayat Scan:

⚠️ **Hanya scan yang sudah synced** yang bisa dihapus.

1. Tahan tekan (long press) pada scan yang ingin dihapus
2. Klik tombol **"Hapus"** (🗑️) yang muncul
3. Konfirmasi penghapusan

## 9. OFFLINE MODE

Aplikasi tetap bisa digunakan TANPA internet.

### Cara Kerja Offline:

1. **Tanpa Internet:**
   - Scan akan tetap berjalan
   - Data disimpan di HP (local database)
   - Status: ⏳ **Pending**

2. **Dengan Internet:**
   - Aplikasi otomatis sinkronisasi
   - Data pending dikirim ke server
   - Status: ✅ **Synced**

### Sync Otomatis:

Aplikasi akan sinkronisasi otomatis saat:
- ✅ HP terhubung ke internet
- ✅ HP sedang di-charge
- ✅ Setiap 15 menit

### Sync Manual:

Jika ingin sync segera:

1. Buka menu **"Settings"**
2. Klik **"Sync Data"**
3. Tunggu proses sync selesai

### Cek Status Sync:

- Buka menu **"Log"**
- Lihat icon status:
  - ✅ = Sudah synced
  - ⏳ = Belum synced (offline)

## 10. TROUBLESHOOTING

### Masalah 1: Kamera Tidak Bisa Dibuka

**Penyebab:** Izin kamera tidak diberikan

**Solusi:**
1. Buka **Settings** HP
2. Cari **"Apps"** → **"Pengamanan"**
3. Klik **"Permissions"**
4. Aktifkan **"Camera"**

---

### Masalah 2: QR Code Tidak Terdeteksi

**Penyebab:** Karena beberapa hal:
- QR code terlalu kecil/besar
- Pencahayaan kurang
- QR code rusak

**Solusi:**
- Dekatkan/jauhkan kamera dari QR
- Pindah ke tempat yang lebih terang
- Pastikan QR code tidak rusak

---

### Masalah 3: Scan Gagal Terus

**Penyebab:** QR code tidak valid atau tidak aktif

**Solusi:**
- Cek apakah QR sudah terdaftar di sistem
- Hubungi admin untuk validasi QR
- Pastikan masa berlaku QR masih aktif

---

### Masalah 4: Data Tidak Sinkron

**Penyebab:** Tidak ada koneksi internet

**Solusi:**
1. Cek koneksi internet
2. Buka **Settings** → **Sync Data**
3. Tunggu hingga selesai

---

### Masalah 5: Lupa PIN

**Solusi:**
- Hubungi admin sistem
- PIN sama untuk semua petugas (global)

---

### Masalah 6: Aplikasi Force Close

**Solusi:**
1. Tutup aplikasi sepenuhnya
2. Buka kembali aplikasi
3. Jika masih berlanjut, restart HP

---

### Masalah 7: Petugas/Pos Tidak Muncul

**Penyebab:** Data belum disinkronkan

**Solusi:**
1. Buka **Settings**
2. Klik **"Sync Data"**
3. Tunggu hingga selesai
4. Coba lagi memilih petugas/pos

## 11. TIPS PENGGUNAAN

### 💡 Saat Bertugas:

1. **Selalu cek petugas dan pos** sebelum mulai bertugas
2. **Scan dengan hati-hati** - pastikan tipe scan benar
3. **Cek riwayat** secara berkala untuk memastikan scan tersimpan
4. **Sync data** setiap berganti shift untuk data terbaru

### 💡 Menjaga Baterai:

1. **Matikan GPS** jika tidak diperlukan
2. **Turunkan kecerahan layar** untuk hemat baterai
3. **Charge saat ada kesempatan** terutama saat istirahat

### 💡 Jika Ada Masalah:

1. **Catat error** yang muncul
2. **Screenshot pesan error** jika ada
3. **Laporkan ke admin** segera
4. **Jangan hapus aplikasi** sebelum konsultasi

## 12. ALUR KERJA PETUGAS

### Sebelum Bertugas:

```
1. Buka aplikasi
2. Login dengan PIN
3. Pilih petugas (nama Anda)
4. Pilih pos tempat bertugas
5. Klik "Simpan"
6. Klik "Sync Data" untuk data terbaru
```

### Saat Bertugas:

```
1. Ada warga/tamu datang
2. Minta tunjukkan QR code
3. Buka aplikasi → tab "Scan"
4. Arahkan kamera ke QR code
5. Tunggu QR terdeteksi
6. Pilih tipe scan (MASUK/KELUAR)
7. Klik "SCAN"
8. Tunggu notifikasi sukses
```

### Saat Berganti Shift:

```
1. Buka menu "Log"
2. Cek semua scan sudah synced (✅)
3. Jika ada pending (⏳), hubungkan ke internet
4. Buka "Settings" → "Sync Data"
5. Logout dan serahkan ke petugas berikutnya
```
